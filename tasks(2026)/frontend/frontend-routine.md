# Frontend Routine

## 学习目的

Python 前端的生态远不如前端主流框架。

但它仍然可以用于快速验证原型，尤其是 AI 相关的演示原型。

## 学习内容

[前端从入门到如土](https://github.com/west2-online/learn-frontend)

Gradio / Streamlit

## 作业

你可以使用 Python 前端框架完成前端组相关作业来检验学习效果。这些作业不强制提交，但欢迎提交展示并参与答辩。

更推荐你直接尝试前端组或美术组考核。在当前 AI 时代，虽然生成式工具降低了入门门槛，但高质量的审美与交互设计仍然高度依赖人的判断。
